# 高度仿真微信图片选择功能
模仿微信图片选择功能，主要分为以下三个模块：
* 使用任务列表，线程池，图片异步压缩及加载，图片缓存管理等自定义ImageLoader（核心）
* 使用GridView显示图片
* 使用PopupWindow切换图片文件目录
